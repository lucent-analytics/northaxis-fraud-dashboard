
# Operation Clearwater — NorthAxis Bank Fraud Dashboard

## Project Overview
A full end-to-end transaction fraud detection and risk intelligence project 
for NorthAxis Bank, analyzing 195,276 transactions across Jan–Sep 2024.

## Live Dashboard
View the interactive dashboard here:
[NorthAxis Fraud Dashboard](https://lucent-analytics.github.io/northaxis-fraud-dashboard/northaxis_fraud_dashboard.html)

## Tools Used
- MySQL — data storage and querying
- SQL — fraud analysis across 6 analytical tasks
- Power BI — interactive dashboard
- Python / ReportLab — executive PDF report

## Project Structure
| File | Description |
|---|---|
| `northaxis_fraud_dashboard.html` | Interactive 3-page fraud dashboard |
| `NorthAxis_Executive_Risk_Report.pdf` | Board-ready executive report |
| `task1_baseline_kpis.sql` | Transaction overview & KPIs |
| `task2_anomaly_detection.sql` | Anomaly detection & velocity checks |
| `task3_customer_risk_profiling.sql` | Customer risk profiling |
| `task4_merchant_channel_risk.sql` | Merchant & channel risk scoring |
| `task5_fraud_risk_scoring_optimised.sql` | Fraud risk scoring model |

## Key Findings
- Estimated fraud exposure: $2.3M+
- 312 customers flagged as CRITICAL tier
- Mobile and Web channels showed highest fraud rates
- Q3 2024 showed a 340% spike in suspicious activity
